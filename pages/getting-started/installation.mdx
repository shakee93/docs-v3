## Requirements

*   WordPress version 5.0 or Above
*   PHP version 6.0 or Above
*   A working WP-Cron(Link)

## Installing RapidLoad

*   Search RapidLoad in the WordPress Plugins Directory. 
*   Click “_Install Now_,” and the plugin will be installed. 
*   Click “_Activate_,” and you’ll be guided through the onboarding process.

## Updates & Rollback

**How to Update**

*   Go to the Plugins page on WordPress 
*   Locate RapidLoad.
*   Simply click the “_Update Now_” button to apply the latest update.

  **_If an update is available, you’ll see a notification, as shown in the screenshot._**

### Rollback

*   Install this rollback plugin(plugin link). 
*   Select and revert to the version you want.

<Callout type="warning" emoji=" ">
  **_You can view the changelog here(Link) to pick the appropriate version._**
</Callout>

## Uninstalling RapidLoad

*   Navigate to the Plugins section in WordPress and locate RapidLoad. 
*   Click “_Deactivate._” 
*   A pop-up will request feedback please provide it to help us improve RapidLoad.

<Callout type="warning" emoji=" ">
  **_Once deactivated, all page-specific RapidLoad configurations will be cleared, though global settings will remain intact._**
</Callout>
