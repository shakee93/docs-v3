**This guide provides an overview of how to navigate the RapidLoad dashboard effectively.**

**Performance**

  

(img)
The Performance section of the RapidLoad optimizer includes the following parts:

*   **Device Mode** (img): Switch between desktop and mobile modes to configure optimizations for the selected device.
*   **Analyze** (img): Click this button to run a page speed analysis.
*   **Preview** (img): View a preview of the page after RapidLoad optimizations are applied. \[Learn more\]
*   **URL and Date** (img): Displays the currently selected URL for optimization and the date of the latest performance report.

  

**Speed Settings**

  

(img)

  

In this section, you can configure all aspects of the RapidLoad plugin.

*   **Performance Gears: Select a gear to activate specific optimization features immediately. \[Learn more\]**
*   **Customize Settings** (img)**: Here you can adjust RapidLoad settings as desired, rather than relying solely on preset gears.\[Learn more\]**
*   **Audits: View site audits, separated into** **_Opportunities_** **and** **_Diagnostics_****, which impact your score. Passed audits are also listed here. \[Learn more\]**
