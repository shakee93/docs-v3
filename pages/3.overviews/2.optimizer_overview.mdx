**This guide provides an overview of how to navigate the RapidLoad dashboard effectively.**

**Performance**
  
<img style={{ marginTop: '24px' }} 
src="/images/rl-docs-images/optimizer/performance.png" alt="Update Plugin" />

The Performance section of the RapidLoad optimizer includes the following parts:

*   **Device Mode** 

<img style={{ marginTop: '24px' }} 
src="/images/rl-docs-images/optimizer/device-mode.png" alt="Update Plugin" />

Switch between desktop and mobile modes to configure optimizations for the selected device.

*   **Analyze** 

<img style={{ marginTop: '24px' }} 
src="/images/rl-docs-images/optimizer/analyze.png" alt="Update Plugin" />

Click this button to run a page speed analysis.

*   **Preview** 

<img style={{ marginTop: '24px' }} 
src="/images/rl-docs-images/optimizer/preview.png" alt="Update Plugin" />

View a preview of the page after RapidLoad optimizations are applied. \[Learn more\]

*   **URL and Date**

<img style={{ marginTop: '24px' }} 
src="/images/rl-docs-images/optimizer/URL-and-Date.png" alt="Update Plugin" />


Displays the currently selected URL for optimization and the date of the latest performance report.

  

**Speed Settings**

  
<img style={{ marginTop: '24px' }} 
src="/images/rl-docs-images/optimizer/settings.png" alt="Update Plugin" />


In this section, you can configure all aspects of the RapidLoad plugin.

*   **Performance Gears: Select a gear to activate specific optimization features immediately. \[Learn more\]**
*   **Customize Settings** (img)**: Here you can adjust RapidLoad settings as desired, rather than relying solely on preset gears.\[Learn more\]**
*   **Audits: View site audits, separated into** **_Opportunities_** **and** **_Diagnostics_****, which impact your score. Passed audits are also listed here. \[Learn more\]**
